A "Set It and Forget It" Solution

SapeBridge is a simple, secure service built to replace the Amadeus Stand-Alone Print Emulator (SAPE). We handle all the technical complexity of SFTP so you can keep your familiar workflow.

How It Works:

  1. 1. We give you secure SFTP credentials for Amadeus.
  2. 2. Amadeus drops files into our secure system.
  3. 3. Our lightweight Windows client automatically downloads the files to your designated local folder.
  4. 4. Your back-office system processes the files just like it always has.

Frequently Asked Questions

What is Amadeus's new SFTP requirement for AIR files?

Amadeus now requires travel agencies to use SFTP for delivering AIR and/or IMR files, replacing the previous SAPE polling method. This change means agencies must adapt to a new technical process for file delivery.

Do I need to set up and host my own SFTP server?

No. SapeBridge eliminates the need for agencies to set up, host, or maintain an SFTP server. Our solution connects Amadeus's SFTP system directly to your local computers, saving you time and technical headaches.

Will my agency's workflow change with SapeBridge?

No. Your agents can continue generating AIR files as usual by issuing the INV command in Amadeus. SapeBridge handles the new SFTP delivery process behind the scenes.

Is SapeBridge secure?

Yes. SapeBridge uses end-to-end encryption and enterprise-grade reliability to ensure your data is protected throughout the file delivery process.

How quickly can I get started?

Once ready, onboarding is quick and easy. You can be up and running with SapeBridge in just a few steps, without needing IT staff or server management.

Who is SapeBridge for?

SapeBridge is designed for travel agencies using Amadeus and AIR/IMR files who want a simple, secure solution for the new SFTP requirement.
